# My dotfiles
This repo contains the dotfiles for my system.

Managed by [Stow](https://www.gnu.org/software/stow/).

## Dependencies
- stow
- git

## Setup
- Install Dependencies
- Checkout the dotfiles repo

```
cd ~
git clone https://github.com/nahilrasheed/dotfiles.git
cd dotfiles
```
- use stow to create symlinks for each program.
```
stow program1 program2
```
